# 天翼云盘驱动使用指南

## 快速开始

### 1. 在数据库中添加存储策略

```sql
INSERT INTO storage_policies (name, type, access_key, secret_key, is_private, settings, created_at, updated_at)
VALUES (
  '天翼云盘',
  'cloud189',
  '你的手机号或邮箱',
  '你的密码',
  1,
  '{}',
  NOW(),
  NOW()
);
```

### 2. 在管理面板中配置

1. 登录 Cloudreve 管理面板
2. 进入"存储策略"页面
3. 点击"添加存储策略"
4. 填写以下信息：
   - 策略名称: 天翼云盘
   - 存储类型: cloud189
   - AccessKey: 你的天翼云盘账号
   - SecretKey: 你的天翼云盘密码
   - 是否私有: 是

### 3. 测试上传

上传一个小文件测试驱动是否正常工作。

## 常见问题

### Q: 登录失败怎么办？
A: 请检查账号密码是否正确，确保账号没有开启二次验证。

### Q: 上传失败怎么办？
A: 检查日志文件，查看具体错误信息。可能是网络问题或账号权限问题。

### Q: 支持哪些功能？
A: 目前支持文件上传、下载、删除和列表功能。
